강의 개요

4주 실전 바이브코딩 강의 시나리오 요약

목표: 4주 프로젝트로 현업 즉시 적용 가능한 스택 습득

결과물: 포트폴리오로 활용 가능한 데스크톱 애플리케이션

핵심 가치: 배우면 바로 써먹는 실용적 스킬

목차: 동기부여 포인트 주차 구성

매주 강의 시작 동기부여 포인트

  • 오늘 배우는 내용의 실무 사용 사례 제시
  • 포트폴리오 임팩트 강조
  • 막히는 포인트 선제 대응 팁

주차 구성

  1. 1주차: 일렉트론 기초, 파일시스템/컴포넌트
  2. 2주차: 외부 API 연동(Dify, Scrapper), 에러 핸들링
  3. 3주차: YouTube 데이터 분석 및 시각화 대시보드
  4. 4주차: 구글 OAuth, 로그인 플로우, 인증 기반 기능
